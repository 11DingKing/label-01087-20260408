# 智能停车场管理系统 - 质检手册

## 1. 测试环境准备

### 1.1 系统要求
- 操作系统：Linux / macOS / Windows (WSL)
- 编译器：g++ 或 clang++ (支持 C++17)
- Docker（可选，用于容器化测试）

### 1.2 编译项目

```bash
# 进入项目目录
cd backend

# 创建构建目录
mkdir -p build

# 编译主程序
g++ -std=c++17 -Wall -I include \
    src/parking_lot.cpp \
    src/main.cpp \
    -o build/parking_system

# 编译测试程序
g++ -std=c++17 -Wall -I include \
    src/parking_lot.cpp \
    src/tests.cpp \
    -o build/run_tests
```

## 4. 日志检查

### 4.1 日志文件位置

```
backend/data/system.log
```

### 4.2 日志格式

```
[2026-01-31 10:30:45] [INFO] [parking_lot.cpp:45] 车辆 京A12345 入场，停放于 A01
[2026-01-31 10:35:20] [INFO] [parking_lot.cpp:78] 车辆 京A12345 出场，费用: 5.00 元
[2026-01-31 10:36:00] [WARN] [parking_lot.cpp:52] 车辆 京A12345 已在停车场内
```

### 4.3 日志级别说明

| 级别 | 说明 | 示例 |
|------|------|------|
| DEBUG | 调试信息 | 变量值、流程跟踪 |
| INFO | 正常操作 | 入场、出场、注册 |
| WARN | 警告信息 | 重复入场、车位已满 |
| ERROR | 错误信息 | 数据异常、系统错误 |
